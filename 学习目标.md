**项目 1：智能项目协作与任务管理系统**

类似轻量版 Jira / 飞书项目 / Trello。

核心功能：

- 组织、项目、成员、角色权限
- 任务看板：待办、进行中、已完成
- 任务指派、优先级、截止时间、标签
- 评论、附件上传、操作日志
- 消息通知、WebSocket 实时提醒

- 后台管理：用户、角色、菜单、权限

能练到的能力：

- RBAC 权限模型
- 前后端分离架构
- 复杂表单、表格、弹窗、详情页
- 实时通信
- 文件上传
- 数据统计
- 企业后台开发规范

简历写法：

> 基于 Vue3 + Spring Boot 开发项目协作平台，实现项目管理、任务看板、RBAC 权限控制、WebSocket 实时通知、文件上传与数据统计模块，支持 Docker Compose 一键部署。

这个项目最像真实公司内部系统，非常适合找 Java / 全栈 / 后台管理类岗位。

**AI 辅助开发方式**

你后续开发时，可以固定使用这套节奏：

1. 先让 AI 拆需求：这个模块有哪些页面、接口、表结构。
2. 再让 AI 生成初版代码：Controller、Service、Mapper、DTO、VO。
3. 自己审查代码：命名、权限、异常、事务、边界条件。
4. 让 AI 帮你补测试和优化。
5. 每完成一个模块，写一次 Git commit。
6. 最后让 AI 帮你整理 README 和简历描述。
