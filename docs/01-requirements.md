# 需求文档：智能项目协作与任务管理系统

## 1. 项目定位

CodeCollab 是一个面向团队协作的项目管理系统，核心目标是帮助团队围绕项目、任务、成员、讨论和进度进行协同。

它不是一个只演示 CRUD 的练习项目，而是一个贴近企业后台系统的全栈训练项目。

## 2. 用户角色

### 平台管理员

- 管理系统用户
- 管理角色和权限
- 查看系统运行数据

### 组织管理员

- 创建和维护组织
- 邀请成员
- 分配组织角色
- 管理组织下的项目

### 项目负责人

- 创建项目任务
- 分配任务负责人
- 查看项目进度
- 管理项目成员

### 普通成员

- 查看参与项目
- 处理被分配任务
- 评论任务
- 上传附件
- 接收通知

## 3. 核心模块

### 认证与用户模块

- 登录
- 退出
- 刷新 Token
- 获取当前用户信息
- 修改个人资料
- 修改密码
- 头像上传

### RBAC 权限模块

- 用户管理
- 角色管理
- 菜单管理
- 权限标识管理
- 前端动态路由
- 后端接口鉴权
- 按钮级权限控制

### 组织模块

- 创建组织
- 更新组织信息
- 邀请成员加入
- 移除成员
- 成员角色分配

### 项目模块

- 创建项目
- 编辑项目
- 归档项目
- 项目成员管理
- 项目概览
- 项目统计

### 任务模块

- 创建任务
- 编辑任务
- 删除任务
- 指派负责人
- 设置优先级
- 设置截止时间
- 添加标签
- 子任务
- 任务状态流转
- 任务操作日志

### 看板模块

- 待处理、进行中、已完成等列
- 拖拽切换状态
- 拖拽排序
- 按负责人筛选
- 按优先级筛选
- 按截止时间筛选

### 评论与附件模块

- 任务评论
- 评论回复
- 附件上传
- 图片预览
- 文件下载
- 文件权限校验

### 通知模块

- 被指派任务通知
- 评论通知
- 截止时间提醒
- WebSocket 实时推送
- 已读/未读管理

### 数据看板模块

- 项目任务总数
- 任务状态分布
- 成员任务分布
- 任务完成率
- 近 7 天任务趋势

## 4. 非功能需求

- 统一接口响应格式
- 统一异常处理
- 统一鉴权机制
- 关键接口参数校验
- 核心操作写入操作日志
- 支持 Docker Compose 本地部署
- 保留扩展为 SaaS 多租户的空间

## 5. 简历亮点

项目完成后可以写成：

> 基于 Vue 3 + TypeScript + Spring Boot 开发智能项目协作系统，实现用户认证、RBAC 权限控制、组织与项目管理、任务看板、评论附件、WebSocket 实时通知和项目数据统计。后端采用 Spring Security、JWT、MyBatis-Plus、Redis、MinIO，前端基于 Pinia、Vue Router、Element Plus 和 ECharts 构建，支持 Docker Compose 一键部署。
